## 📌 Overview

**Ty-SMS-Apk** is a professional SMS bombing app designed for security testing and educational purposes. It utilizes 60+ different APIs to send SMS messages to target numbers. The tool is **zip apk** and works on **Android**, and any other Linux distribution.

### ✨ Features

| Feature | Description |
|---------|-------------|
| 🚀 **60+ APIs** | Multiple SMS APIs for maximum effectiveness |
| 💻 **Cross-Platform** | Works on Termux, Kali, Ubuntu, Parrot, BlackArch |
| 🎨 **Professional UI** | Colorful animated interface with progress bars |
| ⚡ **High Speed** | Optimized for fast SMS delivery |
| 📊 **Real-time Stats** | Live success/failure tracking |
| 🛡️ **Anti-Ban** | Random User-Agents and headers rotation |

## 📲 Install on Termux

```bash
pkg update -y
pkg install zip unzip -y
pkg install git -y
git clone https://github.com/tyranroot/Ty-SMS-Apk.git
cd Ty-SMS-Apk
unzip Ty-SMS.zip
mkdir /sdcard/Ty-SMS-Apk
cp Ty-SMS.apk /sdcard/Ty-SMS-Apk
````
```
Now go to your file manager and check the Ty-SMS-Apk file... Then Install the SMS Bomber Apk
```
